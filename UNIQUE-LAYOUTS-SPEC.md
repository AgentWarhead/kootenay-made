# Unique Layout Spec — 16 Industry-Specific Demo Pages

## The Problem
All 16 demos currently follow the same template: Nav → Hero → Trust Bar → Services → How It Works → Gallery (1 big + 3 small) → Before/After → Testimonials → FAQ → About → Contact → Footer. Cookie cutter is what we sell AGAINST.

## The Solution
Each demo gets a unique section layout, gallery style, and content priority that reflects how REAL top websites in that industry are designed.

---

## 1. AUTOMOTIVE (Iron Horse Garage)
**Vibe:** Dark, powerful, chrome accents. Think premium garage, not jiffy lube.
**Unique layout elements:**
- Hero: Full-bleed dark with chrome gradient text, speedometer animated stats
- **Gallery: Before/after vehicle transformations** — side-by-side pairs (not 1+3 grid). Show "Brought in rusted" → "Left showroom ready"
- **Services: Icon grid with hover-reveal pricing** — not card stack
- Add: **"Currently in the Shop"** section — fake status board showing 3 vehicles in progress (builds urgency/credibility)
- Testimonials: Single scrolling ticker strip (race-stripe style), not cards

## Implementation Priority
Each demo needs these sections reimagined:
1. **Gallery/Showcase** — the most cookie-cutter section, needs the most variation
2. **Services** — many industries don't present services as cards (menus, schedules, directories)
3. **Unique industry section** — the one thing that makes each demo unmistakably for THAT industry
4. **Before/After (LiveRedesign)** — after state must match the new unique design language

## Rules
- NO two demos should share the same gallery layout
- NO two demos should share the same services presentation format
- Each demo must have at least ONE section that NO other demo has
- Maintain existing heading fonts, color palettes, and decorative elements per demo
- All content is sample/placeholder — clearly labeled as such
- Mobile-first responsive on everything
